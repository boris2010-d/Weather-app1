# Weather App 🌤️

A simple and modern weather application built with **HTML**, **CSS**, and **JavaScript**.  
The app uses the **OpenWeatherMap API** to fetch real‑time weather data for any city.

---

## 🚀 Features

- 🔍 Search for any city
- 🌡️ Display temperature, humidity, wind speed
- 🌤️ Weather description and icon
- 🎨 Dynamic background based on weather conditions
- 😀 Mood emoji that changes depending on the weather
- ⚡ Loading and error handling
- 📱 Fully responsive design

---

## 🖼️ Dynamic UI Elements

### 🌈 Background changes based on weather ID:
- Thunderstorm → dark stormy gradient  
- Rain → blue/gray rainy gradient  
- Snow → light blue snowy gradient  
- Clear sky → warm sunny gradient  
- Clouds → gray cloudy gradient  
- Fog → soft foggy gradient  
- Night fallback → dark blue gradient  

### 😀 Mood Emoji
Emoji updates automatically:
- ⛈️ Thunderstorm  
- 🌦️ Drizzle  
- 🌧️ Rain  
- ❄️ Snow  
- 🌫️ Fog  
- 😎 Clear sky  
- ☁️ Clouds

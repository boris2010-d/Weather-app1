# 🌤️ Weather App

A clean and modern weather application built with **HTML**, **CSS**, and **JavaScript**, using the **OpenWeatherMap API**.

## 🚀 Features
- 🌍 Search for any city
- 🌡️ Temperature, humidity, wind speed
- 🌤️ Weather description + official icon
- 🎨 Dynamic background based on weather conditions
- 😀 Mood emoji that updates automatically
- ⚠️ Smart weather alerts (storm, rain, snow, fog, wind, heat, cold)
- 🕒 Hourly forecast for today
- 📅 5‑day forecast with clickable cards
- 🔁 °C / °F unit toggle
- 📱 Fully responsive design

## 🛠️ Technologies
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
- Google Fonts

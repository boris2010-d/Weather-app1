🌤️ Weather App — Modern Weather Dashboard
A modern, responsive and feature‑rich weather application built with HTML, CSS, and JavaScript.
The app uses the OpenWeatherMap API to fetch real‑time weather data, 5‑day forecasts, hourly forecasts, and provides a clean, intuitive UI.

🚀 Features
🔍 City Search
Search for any city worldwide and instantly get weather data.

🌡️ Current Weather
Displays:

Temperature

Humidity

Wind speed

Weather description

Official OpenWeather icons

🎨 Dynamic Background
Background changes automatically based on weather conditions:

⛈️ Thunderstorm → dark stormy gradient

🌧️ Rain → blue/gray rainy gradient

❄️ Snow → light blue snowy gradient

☀️ Clear sky → warm sunny gradient

☁️ Clouds → gray cloudy gradient

🌫️ Fog → soft foggy gradient

🌙 Night → dark blue gradient

😀 Mood Emoji
Emoji updates based on weather:

⛈️ Thunderstorm

🌦️ Drizzle

🌧️ Rain

❄️ Snow

🌫️ Fog

😎 Clear sky

☁️ Clouds

⚠️ Weather Alerts
Smart alerts for:

Storm

Heavy rain

Snow

Fog

Strong wind

Extreme heat

Extreme cold

🕒 Hourly Forecast
Shows hourly temperature, icon, and time for the current day.

📅 5‑Day Forecast
Displays 5 clickable forecast cards.
Each card opens a detailed day page with:

Full weather info

Hourly breakdown

Large icon

Temperature

Humidity

Wind

Back button

🔁 Unit Toggle (°C / °F)
Switch between Celsius and Fahrenheit instantly.

🕘 Loading & Error Handling
Smooth loading indicator and clean error messages.

📱 Fully Responsive
Optimized for desktop, tablet, and mobile.
